# 이진서 내전 계산기

이터널 리턴 내전 운영용 정적 웹 계산기입니다. 참가자 신청, 시즌 랭크 조회, 역할군 3픽, 블라인드 팀 편성, 팀장 지정, 리플레이 코드, 밴 기록, 4경기 점수 계산, JSON 백업/복원을 한 화면에서 관리합니다.

## 실행

브라우저에서 `index.html`을 열면 바로 사용할 수 있습니다. 로컬 서버로 확인하려면 아래처럼 실행합니다.

```powershell
python -m http.server 5174 --bind 127.0.0.1
```

접속 주소는 `http://127.0.0.1:5174`입니다.

## GitHub Pages 배포

저장소에 아래 파일들을 그대로 올리면 정적 사이트로 동작합니다.

- `index.html`
- `style.css`
- `app.js`
- `2.PNG`
- `3.PNG`
- `4.PNG`
- `5.png`

GitHub 저장소의 `Settings > Pages`에서 배포 브랜치를 선택하면 사이트 주소가 생성됩니다.

## API 설정

랭크 자동 조회는 Eternal Return Open API 키가 필요합니다. 사이트 상단의 API 설정에 키를 입력하고 저장하면 브라우저 `localStorage`에만 저장됩니다.

자동 조회가 실패하거나 API 시즌 ID가 맞지 않을 때는 수동 MMR/랭킹을 입력해서 신청을 완료할 수 있습니다.

## 백업

상단의 `백업` 버튼으로 현재 참가자, 팀, 점수, 리플레이 코드, 밴 기록을 JSON 파일로 내려받습니다. `복원` 버튼으로 다시 불러올 수 있습니다.

CSV 내보내기는 경기 결과 공유나 시트 보관용입니다.
